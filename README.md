# AKT Trust Website

A responsive and modern trust-based website developed for **AKT Trust** to support community services, donations, and volunteer engagement.

## 🌟 Features

- Responsive modern design
- About Us section
- Services section
- Contact section
- Apply for Services form
- Donation section with payment information
- Smooth navigation scrolling
- Mobile-friendly layout
- Interactive buttons and UI

## 💝 Purpose

This website was created to help AKT Trust:
- Connect with the community
- Accept donations
- Allow users to apply for services
- Increase awareness of trust activities

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
